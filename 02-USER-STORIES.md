# 1. ПЕРСОНЫ

## Persona 1: Alex — NFT Project Founder

```yaml
name: Alex
role: Founder NFT-проекта "CryptoBeasts"
age: 28
location: Remote (EU timezone)
tech_level: High

goals:
  - Автоматизировать рутинный маркетинг
  - Реагировать на события (mint, sale) мгновенно
  - Поддерживать активность в 3+ каналах
  - Понимать, что работает

frustrations:
  - Трачу 3 часа в день на посты
  - Каждый канал требует разный формат
  - Пропускаю важные моменты (редкий mint)
  - Не понимаю ROI маркетинга

tools_now:
  - Twitter вручную
  - Discord боты (базовые)
  - Google Sheets для планирования
```

## Persona 2: Marina — SMM Manager

```yaml
name: Marina
role: SMM-менеджер в агентстве
age: 32
location: Москва
tech_level: Medium

goals:
  - Управлять 5-10 клиентами одновременно
  - Экономить время на создании контента

frustrations:
  - Постоянное переключение между аккаунтами

tools_now:
```
